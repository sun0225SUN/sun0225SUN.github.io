什么是HTML？

超文本标记语言（英语：HyperText Markup Language，简称：HTML）

  • HTML 指的是超文本标记语言: HyperText Markup Language
  • HTML 不是一种编程语言，而是一种标记语言
  • 标记语言是一套标记标签 (markup tag)
  • HTML 使用标记标签来描述网页
  • HTML 文档包含了HTML 标签及文本内容
  • HTML文档也叫做 web 页面

WEB 发展史 -> 传送门

HTML 发展史 -> 传送门

CSS 发展史 -> 传送门

HTML文件结构

标签

  • HTML 标签是由尖括号包围的关键词，比如 <html>
  • HTML 标签通常是成对出现的，比如 <b> 和 </b>
  • 标签对中的第一个标签是开始标签，第二个标签是结束标签

元素

当用一组HTML标签将一段文字包含在中间时，这段文字与包含文字的HTML标签 被称作一个元素

HTML基本标签

<!DOCTYPE> 声明

<!DOCTYPE>声明有助于浏览器中正确显示网页，声明不区分大小写。

元信息标签<meta>

元信息包含的信息是用户看不见的，一般用来定义页面信息的名称、关键字、作者等。编写中卫一般在meta中声明编码格式utf-8或GBK

文件开始标签<html>

文件头部标签<head>

文件标题标签<title>

HTML 标题（Heading）是通过<h1> - <h6> 标签来定义的。

页面主体标签<body>
